# PC Speed Test Web

Static landing page for Firebase Hosting.

Production URL: `https://pcspeedtool.web.app/`

## Quick start

1. Install Firebase CLI if needed:
   `npm install -g firebase-tools`
2. Log in and select the default Firebase project:
   `firebase login`
   `firebase use pcspeedtool`
3. Deploy from this folder:
   `firebase deploy`

## Structure

- `firebase.json`: Firebase Hosting config
- `.firebaserc`: default Firebase project alias (`pcspeedtool`)
- `public/`: static site files
